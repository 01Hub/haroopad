Haroopad

is a tool for creating web-friendly document in markdown editor.

You can also based on the markdown, to create Web documents, blogs, as well as e-mail, the report documents expert.

And then support Windows, Linux, the Mac OS, you get the experience of editing the same document markdown in the desired platform.


Markdown? Learn Haroopad

Features

Support Cross Platform

Haroopad support (Ubuntu, 32,64 bit) Windows7 / 8 (32bit), Mac (32bit), of Linux.

Multimarkdown + Extensions

In addition to the grammar markdown the standard, extended syntax of multi markdown is adding a lightweight markup that you need to edit the document and modern.

Easy Publishing (Comming)

Wordpress, tumblr, also provides output function to the cloud service and blog, such as Google Docs to provide PDF, HTML output function.

Theme, Skin, Component

You can create a more powerful interactive document the My Documents themes and skins, and to provide various components for the markdown view.

Easy Import Contents (Comming)

Most records from the time we produce, are stored YouTube, blog, slide share, in places like Flickr. You can easily be added to My Documents this content.

Social Stream (Comming)

You can tell easily Facebook, Twitter, etc through a document that was created in the Haroopad.
