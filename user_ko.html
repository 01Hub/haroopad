하루패드

는 새로운 문서 도구로 웹 친화적인 문서 작성을 위한 마크다운 에디터입니다.

마크다운을 기반으로 웹 문서, 블로그, 메일뿐만 아니라 전문가적인 리포팅 문서를 작성할 수도 있습니다.

윈도우, 리눅스, 맥 운영체제를 지원하여 여러분이 원하는 플랫폼에서 동일한 마크다운 문서편집 경험을 얻을 수 있습니다.

Main Features

크로스 플랫폼 지원

하루패드는 윈도우 7/8(32bit), 맥(32bit), 리눅스(Ubuntu 32, 64bit) 환경에서 동일한 편집 환경을 제공합니다.

멀티 마크다운과 확장기능

멀티 마크다운과 확장 문법은 마크다운 표준 문법 이외에 모던한 문서 편집에 필요한 경량 마크업을 추가로 제공합니다.

다양한 포맷 변환 지원 (예정)

PDF, HTML, 위키, EPUB, ReStructured Text, RTF 등으로 변환 출력이 가능하여 마크다운으로 작성한 글을 다양한 스크린에서 볼 수 있습니다.

테마 기능 지원

현대적인 에디팅 테마, 마크다운 테마 그리고 코드 구문 강조 기능을 제공하여 내 문서를 더욱 강력하고 가독성이 뛰어난 문서로 만들 수 있습니다.

손쉬운 콘텐츠 임베딩 (예정)

언젠가부터 우리가 생산하는 대부분의 기록은 유튜브, 블로그, 슬라이드쉐어, 플리커와 같은 곳에 저장되어 있습니다. 이런 콘텐츠를 내 문서에 손쉽게 추가할 수 있습니다.

블로깅과 메일링 지원 (예정)

하루패드는 작성된 문서를 보이는 그대로 메일로 보내거나 워드프레스, 텀블러와 같은 콘텐츠 관리 시스템으로 퍼블리싱을 지원합니다.

Up Coming Features

가독성이 뛰어난 메일 보내기

마크다운으로 작성된 웹 문서를 보이는 그대로 상대방이 받아 볼 수 있다면 얼마나 좋을까요?

하루패드에서는 문서에 포함된 헤더, 이미지, 테이블, 인용구등을 보이는 그대로 메일로 보낼 수 있습니다. 심지어 코드 구문 강조까지 가능합니다.

일반적인 메일 클라이언트에서 제공하는 위지윅보다 더욱 쉽고 간편하게 가독성이 뛰어난 메일을 작성하고 보낼 수 있습니다.

하루패드 메일 보내기 기능을 통해 보내는 사람과 받는 사람이 똑같은 생각을 할 수 있게 되고 메일로 발생하던 오해나 실수는 꽤 줄어들 것입니다.

메일 기능에서 착안된 블로그 포스팅

마크다운은 웹과 매우 친화적인 경량의 텍스트 문서를 위한 마크업입니다.

Github 를 통해 널리 알려진 마크다운 포맷은 메일뿐만 아니라 블로그, 개발문서, 위키 등에도 많이 활용될 수 있게 되었습니다.

마크다운은 자체로의 표현법보다 HTML 로 변환되었을 때 더욱 강력합니다.

이러한 확장성을 이용해 마크다운을 지원하는 텀블러 블로깅 기능을 내장했습니다. 하루패드에서 이젠 블로깅도 가능합니다. 참조 "메일을 이용한 포스팅"

향후 좀더 많은 블로그 시스템과의 연동 계획도 가지고 있습니다.

Vision

마크다운은 단순합니다. 하지만 그 만큼 이식성이나 확장성이 강합니다.

하루패드의 목표는 단순합니다. 이 특징을 이용한 웹 친화적인 문서 편집 도구입니다.

하루패드는 기존의 컨텐츠 메니지먼트와 클라우드 시스템의 연동, 메일, 프리젠테이션, 할일 관리, 문서 공동 작업등을 기본으로 문서 편집의 새로운 영역에 연구와 개발을 진행할 예정입니다.

하루패드를 사용해 보시고 편리하고 유용하다 생각되신다면 개발이 유지될 수 있도록 도와주세요.
